# Weignstone Churchil Portfolio

A personal portfolio landing page for **Weignstone Churchil**, a Software Engineering student at Mount Kenya University. The site uses HTML, CSS, and JavaScript to create a cyberpunk-inspired portfolio with animated visual effects and interactive UI elements.

## Features

- Matrix rain background effect.
- Typewriter text animation.
- Animated statistic counters.
- Skill cards with progress animations and hover glitch effects.
- Scroll reveal for content sections.
- Responsive navbar with active section highlighting.
- Custom cursor trail and boot sequence overlay.
- Contact section with call-to-action.

## Project Structure

- `index.html` - Main portfolio page markup.
- `style.css` - Styling and layout for the portfolio.
- `index.js` - Interactive animations and UI behavior.

## Usage

1. Open `index.html` in a web browser.
2. Alternatively, serve the folder using a local web server for the best experience.

### Local development example

Use Python to serve the files from the project directory:

```bash
cd /home/weignstone/Documents/Open\ UNI/PT.open
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
